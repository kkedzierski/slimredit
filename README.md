# Slimreddit 
### The application allow users to view and search posts and comments provided by the simplified reddit API.

### Technologies

***

* HTML
* CSS
* JavaScript
* React
* Redux


### Instalation

***

After pull respository to open project on localhost write in console:  
```npm start```
